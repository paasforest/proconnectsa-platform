## Setup Checklist

### Facebook Business Page
- [ ] Create Facebook account (if needed)
- [ ] Go to facebook.com/pages/create
- [ ] Choose "Business or Brand"
- [ ] Enter page name and category
- [ ] Add description
- [ ] Add website URL
- [ ] Upload profile picture
- [ ] Upload cover photo
- [ ] Add contact information
- [ ] Publish page
- [ ] Add link to website in About section

### LinkedIn Company Page
- [ ] Create LinkedIn account (if needed)
- [ ] Go to linkedin.com/company/setup
- [ ] Enter company name
- [ ] Add industry and company size
- [ ] Add website URL
- [ ] Add company description
- [ ] Upload logo
- [ ] Add specialties
- [ ] Publish page

### Twitter/X Profile
- [ ] Create Twitter account (if needed)
- [ ] Go to twitter.com/settings/profile
- [ ] Add display name
- [ ] Add bio
- [ ] Add website URL
- [ ] Add location
- [ ] Upload profile picture
- [ ] Upload header photo

### Instagram Business Profile
- [ ] Create Instagram account (if needed)
- [ ] Convert to Business Profile (Settings > Account > Switch to Professional Account)
- [ ] Add name and bio
- [ ] Add website URL
- [ ] Add category
- [ ] Upload profile picture
